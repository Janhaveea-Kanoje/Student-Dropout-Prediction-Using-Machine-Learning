Introduction

Business Problem - A high student dropout rate affects the financial stability and reputation of the institution. Analysing this data will help the institution to get insights on the factors responsible for dropout rate and prevent it.

Data source for this model is: https://www.kaggle.com/datasets/thedevastator/higher-education-predictors-of-student-retention/data?select=dataset.csv

We aim to formulate this problem into a Machine Learning task.

param_grid = {
    "SVM": {"C": [0.1, 1, 10], "kernel": ["linear", "rbf"], "gamma": ["scale", "auto"]},
    "Random Forest": {"n_estimators": [50, 100, 200], "max_depth": [10, 20], "min_samples_split": [2, 5, 10]},

}

best_models = {}
for name, model in models.items():
    grid = GridSearchCV(model, param_grid[name], cv=5, scoring="accuracy", n_jobs=-1)
    grid.fit(x_train_pre, y_resampled)
    best_models[name] = grid.best_estimator_

    print(f"\nSignificant parameters for {name}: {grid.best_params_}")
    print("Best accuracy:", grid.best_score_)

Conclusion

This machine learning pipeline provides insightful data to educational institutions to increase student performance and engagement. Debt, course, approved curriculars, age, etc,. being key factors for target determination, institutions should focus and seek for improvemnet in these areas by providing better student support and revised curricular.

There are certain limitations to this model and it is still missclassifying data at a noticable rate which can be fixed in future with better enhancements.

Overall, this model can be deployed as it may give a good-informed analysis and understanding of this data which can be useful to the institutions.
